# Flappy Bird Clone

## What is Flappy Bird Clone?
Flappy Bird Clone is a simple yet addictive game where the player controls a bird and aims to fly between columns of pipes without crashing into them. The game is easy to learn but challenging to master, providing an engaging experience for players of all ages.

## Project Structure
```
Flappy-Bird-Clone/
├── index.html 
├── style.css 
├── script.js 
└── assets/
    └── images/
```
- **index.html**: The main HTML file for the game.
- **style.css**: The CSS file to style the game interface.
- **script.js**: The JavaScript file that contains the game logic.
- **assets/**: A folder containing images and other assets used in the game.

## Getting Started
1. Clone the repository:
   ```
   git clone https://github.com/Quantum-Youssef-Amr/Flappy-bird-clone.git
   ```
2. Navigate to the project directory:
   ```
   cd Flappy-bird-clone
   ```
3. Open `index.html` in your web browser to start playing!

## Features
- Engaging gameplay similar to the original Flappy Bird
- Simple and intuitive controls
- Responsive design for various screen sizes
- Scoring system to track high scores
- Attractive visuals and sound effects

Enjoy playing the Flappy Bird Clone!
